## Key Concepts

### Trading Modes

- **Simulation**: Mock data, risk-free learning environment
- **Paper Trading**: Real market data with simulated execution
- **Live Trading**: Real execution with actual capital

See [Trading Guide](trading-guide.md#trading-modes) for details.

### Core Components

- **Exchange Interface**: Unified API for multiple exchanges
- **AI Agent**: Generates trading signals with confidence scoring
- **Risk Manager**: Position sizing and risk validation
- **Order Executor**: Handles order placement and execution
- **Position Monitor**: Tracks open positions and P&L

See [Core Concepts](concepts.md#architecture) for complete details.

### Risk Management

- Position sizing: Risk-based with capital allocation
- Stop-loss: Automatic loss protection
- Take-profit: Profit target management
- Leverage: Configurable per market type

See [Core Concepts](concepts.md#risk-management) for algorithms.
